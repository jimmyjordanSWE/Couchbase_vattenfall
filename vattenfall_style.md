Rules:
1. Use only official Vattenfall logos from the links above. Do not redraw, stretch, recolor, crop, or approximate the logo.
2. Base the UI on Vattenfall’s official palette:
   - White: #FFFFFF
   - Yellow: #FFDA00
   - Blue: #2071B5
   - Grey: #4E4B48
   - Black: #000000
   Secondary colors if needed:
   - Dark green: #005C63
   - Dark blue: #1E324F
   - Pink: #D1266B
   - Green: #3DC07C
   - Red: #F93B18
   - Purple: #9B62C3
   - Dark purple: #85254B
3. Visual direction: clean, spacious, Nordic, bright, calm, technically confident. Avoid trendy startup styling, visual clutter, glossy effects, and loud gradients.
4. Typography: Vattenfall’s official typeface is Hall. If Hall is not available in the project, state that clearly and use a restrained sans-serif fallback. Do not claim the fallback is the official font.
5. Photography and imagery should feel authentic, daylight-lit, real, and understated. Prefer real people, infrastructure, landscapes, and energy contexts over generic stock imagery.
6. Use Vattenfall Blue as the main structural and CTA color. Use Vattenfall Yellow sparingly as an accent, not as the dominant background.
7. Keep layouts airy with generous whitespace, simple hierarchy, clear copy, and minimal decoration.
8. If any brand choice is uncertain, prefer the official source links over invention.

Implementation workflow:
- Start by defining CSS variables for the official colors.
- Build the layout with a white or very light base.
- Use blue for navigation, links, buttons, and key structure.
- Use yellow only for highlights or emphasis.
- Place the official logo with conservative spacing and without alteration.
- Keep icons and illustration minimal.
- At the end, list:
  - which official links were used
  - whether the official logo was used
  - whether Hall was available or a fallback was used
  - any deviations from strict brand compliance
